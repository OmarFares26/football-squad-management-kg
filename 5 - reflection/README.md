# 5 - Reflection and Service

This section contains the KG-backed service, data-model comparison, and
final learning-outcome reflection.

## Contents

- `src/service_output.py`: queries the GraphML KG for Liverpool decisions
- `src/data_model_comparison.py`: compares CSV, property graph, RDF,
  embeddings, and temporal KG design
- `graphs/squad_management_kg.ttl`: RDF/Turtle export
- `results/`: service and data-model comparison outputs
- `docs/learning_outcomes.md`: final LO evidence and mapping
- `docs/portfolio_notes.md`: project narrative and limitations

Both scripts are executed by the root `run_pipeline.py`.
